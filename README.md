# 🚀 MemoryCam – AI-Powered Object Memory Assistant

MemoryCam is an intelligent iOS application that helps users **remember where they kept their things** using AI-powered visual memory and smart search.

---

## ✨ Features

### 🧠 Smart Memory System
- Save objects with name, location, and notes
- Automatic tag generation
- Timestamp tracking

### 🔍 Intelligent Search
- Search using natural language  
  _Example: "where are my keys"_
- Searches across:
  - Item name
  - Location
  - Notes
  - Tags

### 📸 Camera-Based Memory (Real Device)
- Capture frames of objects
- Extract visual features using Vision framework
- Match objects in real-time

### 🎤 Voice Input
- Voice → Text → Search
- Hands-free recall

### ⚡ Simulator Support
- Text-only memory saving
- Simulated scanning mode
- Fully testable without camera

### 📊 Smart Suggestions
- Recent memories
- Frequently accessed items

---
